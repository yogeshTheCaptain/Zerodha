## Create a conda/virtual enviroment
```
conda create -n zerodha_first_app python=3.13
```

## Python Check
```
conda activate zerodha_first_app

python --version

Python 3.13.5

conda deactivate
```

## Install Requirements
```
pip install -r requirements.txt
```
